 # Parent Guide

## Parent Dashboard
Route: `/parent/dashboard`

Use the dashboard to:
- See overview of your children
- Access payments and daily reports quickly
- Check important notifications

## Track Child
Route: `/parent/track-child`

Use this page to:
- View child profile details
- Track attendance (if available)
- View activities and updates shared by employees

## Register Child
Route: `/parent/register-child`

Use this page to:
- Add/register a new child under your account
- Provide necessary information (name, date of birth, etc.)

Note: Registration may create payment invoices automatically.

## Payments
Route: `/parent/payments`

Use this page to:
- View invoices for each child
- Check status: pending / paid / overdue
- Pay or confirm payment (depending on system)

## Daily Report
Route: `/parent/daily-report`

Use this page to:
- View daily summaries for your child:
  - activities
  - notes
  - attendance info (depending on implementation)

## Notifications
Route: `/notifications`

Parents receive notifications about:
- Medical alerts involving their children
- Payment reminders and confirmations
- Notes or activities added by employees
