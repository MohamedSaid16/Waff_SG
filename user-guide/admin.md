 
# Admin Guide

## Admin Dashboard
Route: `/admin/dashboard`

Use this page to:
- Monitor total children, parents, employees
- See attendance overview
- Review pending payments
- Track classroom capacity usage

## Manage Parents
Route: `/admin/manage-parents`

Use this page to:
- Create parent accounts
- Update parent information
- Delete parent accounts 

## Manage Employees
Route: `/admin/manage-employees`

Use this page to:
- Create employee accounts
- Update employee profiles
- Ensure employees are assigned to classrooms (important for features)

## Manage Children
Route: `/admin/manage-children`

Use this page to:
- View and edit child records
- Assign/reassign children to classrooms

## Manage Payments
Route: `/admin/manage-payments`

Use this page to:
- View all payment records
- Update payment status (pending/paid/overdue)
- Send reminders to parents (if implemented)

## Manage Classes
Route: `/admin/manage-classes`

Use this page to:
- Create/update/delete classrooms
- Set classroom capacity
- Assign teacher/employees
- Balance children across classrooms

## Reports
Route: `/admin/reports`

Use this page to:
- View and generate reports (attendance, payments, activities depending on implementation)

## Capacity Management
Route: `/admin/capacity`

Use this page to:
- Monitor classroom utilization vs capacity
- Detect overfilled/underfilled classrooms

## Notifications
Route: `/notifications`

Admins receive notifications about:
- New medical alerts
- Payment events (optional)
- Important system events
