 
# Setup Guide

This guide explains how to run the **Nursery Management App** locally.

## 1) Requirements

Install:

- **Node.js** v18+ (recommended)
- **npm** or **yarn**
- **MongoDB**
  - Local MongoDB Community Server, or
  - MongoDB Atlas (cloud)

Optional:
- MongoDB Compass (to view/edit data easily)
